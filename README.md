# 🎫 Ticket System

Sistema de gerenciamento de tickets desenvolvido com **Angular 21**, utilizando arquitetura moderna com Standalone Components e SSR.

---

## 🚀 Tecnologias utilizadas

- Angular 21
- TypeScript
- SCSS
- Angular Router
- Server-Side Rendering (SSR)
- Angular CLI

---

## 📦 Pré-requisitos

Antes de rodar o projeto, você precisa ter instalado:

- Node.js (versão LTS recomendada)
- npm ou yarn
- Angular CLI 21

Instalar Angular CLI globalmente:

```bash
npm install -g @angular/cli@21
